# Daily Spin — Album of the Day PWA

Two album recommendations every day: one essential classic, one cult deep cut.
Rate albums, build your collection, filter by genre/rating/type.

## Deploy to GitHub Pages (2 minutes)

1. Go to **github.com/new** and create a repo called `daily-spin`
   - Make it **Public**, check **Add a README**

2. In your new repo, click **Add file → Upload files**
   - Drag in ALL files from this folder:
     - `index.html`
     - `manifest.json`  
     - `sw.js`
     - `icon-192.png`
     - `icon-512.png`
   - Commit the upload

3. Go to **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**
   - Hit **Save**

4. Wait ~60 seconds, then visit:
   ```
   https://YOUR-USERNAME.github.io/daily-spin/
   ```

## Install on Android

1. Open the URL above in **Chrome**
2. You should see an **"Install app"** banner, or tap **⋮ → Install app**
3. It now lives on your home screen as a standalone app

## Install on iOS

1. Open the URL in **Safari**
2. Tap the **Share** button → **Add to Home Screen**

## Features
- Two daily picks (Essential Classic + Cult Deep Cut)
- Track listings with standout tracks highlighted
- Thumbs up/down + 5-star rating system
- Filterable collection (by genre, rating, type)
- Works offline after first load
- All data stored locally on your device
